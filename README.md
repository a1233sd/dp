# AntiPlagiarism API — автоматизированная система проверки учебных работ на заимствования

Проект реализует локальный backend для проверки учебных работ на заимствования в **текстовых** и **программных** документах, с формированием результата и визуальным выделением совпадений.

Сервис специально сделан простым: только FastAPI + Swagger, без Docker и без внешних платных сервисов.

---

## Что реализовано по ТЗ

- Проверка учебных работ на заимствования
- Поддержка двух типов материалов:
  - `text` — текстовые работы
  - `code` — программный код
- Ведение эталонной базы документов (`reference`)
- Запуск проверки входного текста/кода
- Расчёт показателей:
  - `originality_percent`
  - `matched_tokens`
  - `total_tokens`
- Список совпадений с источниками
- Поля для визуального выделения: `start_char`, `end_char`, `fragment`
- Готовый HTML с подсветкой `<mark>` (`highlighted_html`)
- Правила исключений (например, служебные фразы/разделы)

## API

### Документы
- `POST /documents` — добавить документ в базу
- `GET /documents` — список документов (фильтры: `kind`, `content_type`)

### Правила исключений
- `POST /rules/exclusions` — добавить regex-правило исключения
- `GET /rules/exclusions` — получить правила
- `DELETE /rules/exclusions/{rule_id}` — удалить правило

### Проверка и отчёт
- `POST /checks` — запустить проверку
- `GET /checks/{check_id}` — получить сохранённый результат
- `GET /health` — состояние сервиса

---

## Пример процесса проверки

1. Добавьте эталонные документы (`kind=reference`, `content_type=text|code`).
2. При необходимости добавьте правила исключений (служебные фрагменты).
3. Отправьте работу в `POST /checks`.
4. Получите итог: процент оригинальности, совпадения по источникам и HTML с визуальной подсветкой.

- Нет аутентификации/ролей
